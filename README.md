 # 🚀 Crypto Market Analytics 

## 📌 Project Summary

This project delivers an end-to-end cryptocurrency market analysis solution to understand trading behavior, market volatility, and investment trends. It combines data cleaning, SQL analysis, and interactive dashboard visualization to generate actionable market insights.

The goal is to help analysts and traders identify market opportunities, monitor risks, and understand crypto trading patterns.

---

# 🎯 Problem Statement

Cryptocurrency markets are highly volatile and difficult to analyze due to:

- Rapid price fluctuations
- Unpredictable trading volume
- Market crash risks
- Complex trend behavior

👉 This project solves these challenges using data-driven analytics and visualization.

📌 Major crash periods can be identified using percentage drop analysis
